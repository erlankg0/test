## Команды для запуска

### Установка зависимостей

Перед запуском убедитесь, что у вас установлен **Node.js**. Для установки зависимостей выполните команду:

```bash
npm install
```

```bash
npm install dev
```

Просмотр проекта http://localhost:5173/
